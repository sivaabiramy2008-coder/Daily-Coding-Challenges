# Day 12 - LeetCode Solutions

## Problem 1

**Problem Name:** Valid Palindrome

**Platform:** LeetCode

**Difficulty:** Easy

**Language:** Java

**Approach:** Two Pointers + Character Validation

**Time Complexity:** O(n)

**Space Complexity:** O(1)

## Problem Statement

Given a string `s`, determine whether it is a palindrome after converting all uppercase letters to lowercase and removing all non-alphanumeric characters. Return `true` if it is a palindrome; otherwise, return `false`.

## Problem 2

**Problem Name:** Convert a Number to Hexadecimal

**Platform:** LeetCode

**Difficulty:** Easy

**Language:** Java

**Approach:** Bit Manipulation + Bit Masking + Unsigned Right Shift

**Time Complexity:** O(1)

**Space Complexity:** O(1)

## Problem Statement

Given a 32-bit integer `num`, return its hexadecimal representation as a lowercase string. For negative numbers, use two's complement representation. Do not use any built-in library methods that directly convert integers to hexadecimal.

- ConvertNumberToHexadecimal.java
