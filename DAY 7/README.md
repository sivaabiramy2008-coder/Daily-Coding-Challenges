## Problem 1

**Problem Name:** Palindrome Linked List

**Platform:** LeetCode

**Difficulty:** Easy

**Language:** Java

**Approach:** Fast & Slow Pointers + Reverse Linked List

**Time Complexity:** O(n)

**Space Complexity:** O(1)

## Problem Statement

Given the `head` of a singly linked list, return `true` if it is a palindrome; otherwise, return `false`.
