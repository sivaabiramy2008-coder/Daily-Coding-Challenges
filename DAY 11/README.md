# Day 11 - LeetCode Solutions

## Problem 1

**Problem Name:** Intersection of Two Arrays

**Platform:** LeetCode

**Difficulty:** Easy

**Language:** Java

**Approach:** Nested Loops + Duplicate Check (Without HashSet)

**Time Complexity:** O(n × m + k²)

**Space Complexity:** O(k)

## Problem Statement

Given two integer arrays `nums1` and `nums2`, return an array of their unique intersection elements. Each element in the result must appear only once, and the order of the output does not matter.
