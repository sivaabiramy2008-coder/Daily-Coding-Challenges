# Day 8 - LeetCode Solutions

## Problem 1

**Problem Name:** Minimum Element After Replacement With Digit Sum

**Platform:** LeetCode

**Difficulty:** Easy

**Language:** Java

**Approach:** Array Traversal + Digit Sum

**Time Complexity:** O(n × d)

**Space Complexity:** O(1)

## Problem Statement

Given an integer array `nums`, replace each element with the sum of its digits and return the minimum value after all replacements.
