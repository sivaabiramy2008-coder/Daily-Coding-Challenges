# Day 9 - LeetCode Solutions

## Problem 1

**Problem Name:** Running Sum of 1D Array

**Platform:** LeetCode

**Difficulty:** Easy

**Language:** Java

**Approach:** Prefix Sum (Running Sum)

**Time Complexity:** O(n)

**Space Complexity:** O(1)

## Problem Statement

Given an array `nums`, return the running sum of `nums`. The running sum of an array is defined as the sum of all elements from the beginning of the array up to the current index.
